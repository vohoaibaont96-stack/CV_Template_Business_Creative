# Thư mục ảnh đại diện

Đặt **một ảnh chân dung** của bạn tại đây (`jpg`, `png`, `webp`).

## Cấu hình

Khai báo đường dẫn trong **`src/avatar/config.ts`**:

```ts
src: "/avatar/ten-file-cua-ban.jpg",
```

Hướng dẫn đầy đủ: [`src/avatar/HUONG_DAN_ANH_DAI_DIEN.md`](../../src/avatar/HUONG_DAN_ANH_DAI_DIEN.md)

## Ảnh demo

Ảnh mặc định: `anh_dai_dien.png` — thay bằng ảnh của bạn (giữ tên hoặc đổi `src` trong `src/avatar/config.ts`).
